    • 性能问题

    为了提高用户体现，首要的就是解决浏览器绘制所带了的性能问题。最经典的莫过重绘和回流这两个概念。

    • 重绘：就是页面重新进行绘制，比方说，修改一个元素的背景颜色。
    • 回流：一般来说，浏览器进入页面的时候就已经进行了一次回流，回流其实指的就是页面重新进行排版布局。
