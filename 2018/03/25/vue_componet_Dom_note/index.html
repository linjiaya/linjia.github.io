vue componet Dom 注意事项

发表于 2018-03-25

受到html本身的一些限制，像<ul>、<ol>、<table>、<select>这样的元素里允许包含的的元素有限制，而另一些像<option>这样的元素只能出现在某些特定元素的内部。

方案：使用特殊的 is特性

  • 应当注意，如果使用来自一下来源之一的字符串模板，则没有这些限制:
  1. <script type="text/x-template">
  2. javascript 内敛模板字符串
  3. vue组件

意思是：
这样不可以

但是用is特殊属性可以

或者template模板标签也可以

或者内联模板字符串也行
