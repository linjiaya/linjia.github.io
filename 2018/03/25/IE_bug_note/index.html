IE6 问题

解决ie6表格内外边框堆叠

在html<table>中加上 border="1" cellspacing="0"

解决表格中td中如果没有东西是，会出现没有边框的现象

在td中加上&nbsp;就会有一个无内容有边框的表格出现。

解决IE6、7、8使用JSON.stringify报JSON未定义错误的问题

在通过javascript将对象类型的参数通过JSON.stringify转换成字符串传递时，IE6、7、8会报：”JSON”未定义的错误。我们可以通过在html文件的head头内引入json2.js文件来解决。

引入之后，JSON.stringify就可以在IE6、7、8正常使用了

IE6不支持console.log()函数。

解决IE8不支持background-size属性

background-size时css3的属性，所以ie8不支持。可以采用滤镜来支持。

这样就可以让IE也实现background-size:100%100%的效果了，注意这两个路径要一样，并且是绝对路径。

但是需要全路径,也可以采用backgroundsize.min.htc,这个可以在https://github.com/louisremi/background-size-polyfill拷贝或者下载。

注意: background-size: cover;需要设置，而且不能用100% 100%等的样式。
这样的央视chrome和ie都是兼容的
